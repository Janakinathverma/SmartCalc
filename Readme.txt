# 📱 SmartCalc - Java Android Application

A high-performance, responsive calculator application built using **Java** and **Android Studio**. This project demonstrates the application of core Java logic, event-driven programming, and Material Design principles in a mobile environment.

---

## 🚀 Key Features

- **Robust Arithmetic Engine:** Handles multi-step calculations with operator precedence logic.
- **Responsive UI:** Adaptive layout designed with XML and Material Design for seamless use across different screen sizes.
- **Error Handling:** Built-in validation for edge cases like division by zero and large number overflows.
- **State Management:** Handles configuration changes (e.g., screen rotation) to preserve user input.

---

## 🛠️ Tech Stack

- **Language:** Java (JDK 11+)
- **IDE:** Android Studio
- **UI:** XML / Material Design Components
- **Build System:** Gradle

---

## 🧠 Logic Implementation

As a **5-Star Java Developer**, I focused on ensuring the backend logic of this calculator is scalable and efficient:

1. **Custom Parsing:** Implemented logic to handle operator precedence (PEMDAS) using a stack-based approach or sequential evaluation.
2. **Precision Management:** Leveraged `BigDecimal` for high-precision arithmetic to avoid standard floating-point inaccuracies.
3. **Event Listeners:** Utilized an optimized event-driven model with `View.OnClickListener` to minimize memory overhead.
